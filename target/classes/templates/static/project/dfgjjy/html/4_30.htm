东方国际教育微课堂运营方案

发布：hdjy 浏览：610次

前言：

为完善东方国际教育课程安排，更好的为学生提供优质的教学服务，现开设“精品微课堂”和“培优微课堂”。微课堂是21世纪最先进的教学模式，结合丰富的网络资源及特色的当地师资力量，让每一个孩子都能体验到科学教育、系统教育、创新教育。我校将全心致力于开发微课堂相聚，完善各部分服务项目。

针对人群：

      主要针对凯旋路小学和实验小学以及校区周边范围覆盖内的小学生。

1、对传统大班没有效果的学生。

2、长期在老师家里补课而没有养成自主学习能力的学生。

3、 没有好的学习习惯，无法达到有效的听课效率的学生。     

4、 对无法适应高强对1对1辅导以及无法接受1对1价格的学生。

5、 对适合自身学习环境需求特别大的学生。

“微课堂”介绍：

1、精品微课堂：

精品微课堂主要上课形式为1对6教学，其内容包含有

（1）：每周六至周日2节文化课（每节课90分钟+60分钟全科补漏）。

（2）：每月开设一节英语口语课，西方文化课（每节课60分钟）。

（3）：周一至周天全天自习室开放，安排辅导老师对孩子进行辅导。

（4）：每月开设一节心理辅导课，及时了解孩子内心状况。

（5）：每月组织一次团队合作活动，加强团队动手学习能力。

精品“微课堂”适合小学各个学期段的学生，培养孩子自主学习能力，改善其学习习惯：从而深层挖掘学生潜在学习能力，为后期推动学习成绩的提高做科学完整的铺垫。

2、“培优”微课堂：

培优微课堂主要上课形式为1对3教学，其内容包含有：

（1）：每周六至周日2节文化课（每节课90分钟+60分钟全科补漏）。

（2）：每两周开设2节英语口语课，西方文化课（每节课60）分钟。

（3）：周一至周天全天自习室开放，安排辅导老师对孩子进行学科辅导。

（4）：每月开设一节心理辅导课，及时了解孩子内心状况。

（5）：每月组织一次团队合作活动，加强团队动手学习能力。

（6）：每两周进行学科专项训练，加强孩子学习针对性，初步形成系统性考试技巧，从而达到解题多思维、多方向的效果。

“培优”微课堂针对性较强，适合基础好、学习目标较强的学生。对学生发散思维，高效率、多方向学习有提升效果。

在具体学习科目计划上：

任课老师在前期教学中，首先对孩子在各科的知识掌握能力上做全面的摸底。根据结果作出这个学科的所有详细的计划。教师授课是重点难点梳理，提供同步作业，巩固作业试题，典型例题讲解等。整个辅导方案以查缺补漏，订立计划、提高兴趣、培养学习习惯、方法、巩固提高、提前预习的教学思考路走，辅导前期对整体的知识进行补漏，之后进行全面的提高。

第一阶段：

    与学生先做偶，再做良师。在此阶段主要以启发、鼓励、引导为主,师生双方建立起良好的教学关系，营造一个严谨而宽松的学习氛围。通过授课教师前期和学生的沟通，进行进一步学生全方位情况,学习问题根本原因等，找出知识漏洞，做出针对性的辅导。

第二阶段：

每次课后根据授课内容预留课后作业，少而精，针对性强，而且必须要让孩子保证完成的质量，养成良好的学习习惯和对待作业的态度。解决学生在学校学习期间遇到的问题，帮助学生扫除学习上的障碍。

第三阶段：

在培养学生的学习方法和学习习惯的基础上,务实学科基础,构建整个小学阶段的系统性学习方案。

微课堂开设目的：

1、培养良好的学习习惯，制定每日学习计划，并按计划完成。端正学习态度，增强学习的主动性，快乐高效的学习。

2、规定时间内,学生能认真完成作业。目的在与培养学生良好的学习习惯，打好“效率战”，在作息结合中逐步提高。

3、帮助孩子那排作息时间，让孩子学会“专时专用”，玩要玩得舒心通畅，做作业时要专心致志，决不分心。保证孩子有足够的娱乐和睡眠时间不可忽略。

4、培养孩子要仔细看题目，认真想问题，做作业先是求对、求好，然后才是求快。监督孩子养成认真审题，认真检查的好习惯。

5、培养注意力。提高孩子同外界诱因作斗争的能力，外界的影响是造成孩子粗心大意做错题的根本原因。

小学是人生中最快乐的阶段，为使以后在中学学习中能有一个坚实的基础，因此环境对学生的影响也是不容忽视的，所以应该紧密注意学生的学习及情绪变化，做好相应的疏导工作，让学生能够以充实以及轻松的状态变得自信、乐观，让学习进入一个新的高度。
